# Deskripsi Proyek

Proyek ini merupakan penelitian survei online mengenai tingkat kepuasan mahasiswa terhadap fasilitas ruang kuliah di FMIPA Universitas Mataram menggunakan pendekatan **non-probability sampling**.

Penelitian dilakukan pada mahasiswa:
- Program Studi Statistika Kelas A angkatan 2024
- Program Studi Ilmu Lingkungan Kelas B angkatan 2024

Metode pengambilan sampel yang digunakan adalah **convenience sampling** (kenyamanan), yaitu teknik pengambilan sampel berdasarkan kemudahan peneliti dalam memperoleh responden.

Data diperoleh melalui penyebaran kuesioner online menggunakan Google Form dan dianalisis menggunakan:
- Analisis deskriptif

## Penentuan Jumlah Sampel

Penentuan jumlah minimum sampel menggunakan rumus Slovin dengan tingkat kesalahan (error) sebesar 11%.

### Rumus Slovin

n = N / (1 + N(e²))

Keterangan:
- n = jumlah sampel
- N = jumlah populasi
- e = tingkat kesalahan (error)

### Perhitungan

Diketahui:
- N = 50
- e = 11% = 0.11

Maka:

n = 50 / (1 + 50(0.11²))

n = 50 / (1 + 50(0.0121))

n = 50 / (1 + 0.605)

n = 50 / 1.605

n = 31.15

Berdasarkan hasil perhitungan menggunakan rumus Slovin diperoleh jumlah minimum sampel sebanyak 31 responden. Dalam penelitian ini jumlah responden yang berhasil diperoleh sebanyak 32 responden sehingga telah memenuhi kebutuhan minimum sampel penelitian.

## Sampel Penelitian

## 2. Naive Estimation
Naive estimation digunakan untuk menghitung estimasi langsung berdasarkan hasil sampel tanpa pembobotan.

### Rumus:
P̂ = x / n

Keterangan:
- P̂ = proporsi estimasi
- x = jumlah responden pada kategori tertentu
- n = total responden

---

## 3. Weighted Estimation
Weighted estimation digunakan untuk menyesuaikan distribusi sampel agar lebih mendekati distribusi populasi.

### Rumus:
wi = Proporsi Populasi / Proporsi Sampel

# Uji Validitas dan Reliabilitas

## Uji Validitas
Pengujian validitas dilakukan menggunakan Corrected Item-Total Correlation.

Hasil pengujian menunjukkan bahwa seluruh item pertanyaan memiliki nilai:
- r_hitung > r_tabel

## Uji Reliabilitas
Uji reliabilitas dilakukan menggunakan metode Cronbach Alpha.

Hasil pengujian diperoleh:

Karena nilai Cronbach Alpha > 0.70, maka instrumen penelitian dinyatakan reliabel.
